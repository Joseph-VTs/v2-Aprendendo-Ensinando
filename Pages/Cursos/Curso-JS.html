Curso em Java Script

Java e JavaScript são duas linguagens de programação que compartilham o nome, mas são bastante diferentes em sua utilização e propósito. Java é uma linguagem de programação de propósito geral, utilizada para criar uma ampla gama de aplicações, desde softwares empresariais até jogos e aplicativos Android.
